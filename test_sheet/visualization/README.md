# Visualization Demo Pack

This folder contains demo files for the Performance Analysis module.

## Demo Structure
- 3 classes: Grade 10A, Grade 10B, Grade 10C
- 3 shared exams across all classes:
  - 2026 Placement Test
  - 2026 Midterm Benchmark
  - 2026 Final Mock
- 4 subjects in every file: listening, speaking, reading, writing

## Best Demo Flow
1. Create class `Grade 10A` and upload the three `Grade 10A - ...` files one by one.
2. Repeat for `Grade 10B` and `Grade 10C`.
3. Open Class Analysis for any class to compare multiple exams and show trend/radar/student tracking.
4. Open Grade Dashboard and choose one shared exam like `2026 Midterm Benchmark` to compare all classes side by side.

## Included Files
- 9 class exam files for upload
- `all_classes_history.csv` as a combined reference dataset
